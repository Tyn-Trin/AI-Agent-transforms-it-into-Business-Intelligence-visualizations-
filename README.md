# Odoo AI Chat Agent

**AI Agent ที่ให้ผู้ใช้งานในบริษัทถามข้อมูลจาก Odoo ERP เป็นภาษาคน (ไทย/อังกฤษ) ผ่านหน้าเว็บแชท อนุมัติเอกสาร และกราฟวิเคราะห์ข้อมูลแบบ Real-time กลับมาทันที**

ไม่ต้องรอรายงาน ไม่ต้องไล่หาข้อมูลในเมนู Odoo เอง — แค่ถาม แล้วให้ AI ตัดสินใจว่าจะดึงข้อมูลไหนมา และแสดงผลแบบไหนให้เหมาะสม

## ปัญหาที่เจอ

ผู้ใช้งานที่ไม่ถนัดเทคนิค (ฝ่ายขาย จัดซื้อ บัญชี หรือผู้บริหาร) ต้องการคำตอบเร็วๆ จาก ERP เช่น "เดือนนี้ขายได้เท่าไหร่" "Vendor ไหนส่ง bill เกินราคา PO" "ใครค้างชำระเกินกำหนด" แต่ติดปัญหา:
- ไม่รู้จัก filter, model หรือชื่อ field ของ Odoo
- ขอรายงานแบบกำหนดเองต้องรอ IT/BI ทำให้ เป็นวันๆ


## ทางแก้

AI Agent (ใช้ Claude, Tool Use / Function Calling) ทำหน้าที่เป็นตัวกลางระหว่างผู้ใช้กับ Odoo ผู้ใช้พิมพ์คำถามเป็นภาษาคนผ่านหน้าเว็บแชทหรือ LINE ตัว Agent จะตัดสินใจเองว่าต้องดึงข้อมูลอะไร เรียกใช้ Tool ที่ถูกต้องเพื่อดึงข้อมูลจาก Odoo แบบ Real-time แล้วตอบกลับเป็นภาษาคน — พร้อมแสดงกราฟแบบ interactive อัตโนมัติเมื่อข้อมูลนั้นเหมาะกับการทำกราฟ (อันดับ เทรนด์ เปรียบเทียบ)
